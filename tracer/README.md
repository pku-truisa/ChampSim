This directory contains example tracing utilities that create ChampSim traces. It currently contains:

 - A tracer for use with Intel PIN (see `pin/README.md` for details)
 - A conversion program for CVP traces
 - Memory allocation trace analysis tool (`pin/analyze_malloc.py`)

## Memory Allocation Trace Analysis

The `pin/analyze_malloc.py` tool provides comprehensive analysis of memory allocation traces generated by the PIN tracer. It supports:

- **Size adjustment**: Adjusts allocation sizes to powers of 2 for cache simulation studies
- **Memory tracking**: Monitors active memory objects and peak usage
- **Comparative analysis**: Compares memory usage before and after modifications

For detailed usage instructions, see `pin/README.md`.
